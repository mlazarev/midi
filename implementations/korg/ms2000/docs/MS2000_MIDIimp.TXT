|   Dn   | vv (vv) | --     --   | Channel Pressure        (A.P=A.Touch)     | C |

+--------+---------+-------------+-------------------------------------------+---+

 n  : MIDI Channel = 0 ~ F
 vv : Value

 A.P: Assignable Pedal
 A.S: Assignable Switch
 P.C: Panel Control
 S.C: Sync Control

 ENA = A : Always Enable
       C : Enabled when Global CtrlChg is enabled.
       P : Enabled when Global ProgChg is enabled.
       B : Enabled when Global P.Bend is enabled.
